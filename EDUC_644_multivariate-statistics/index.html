Winter 2023

Preface

This book is a guide for using R and for understanding the procedures in EDUC 644 Applied Multivariate Statistics, which was last offered in 2023. It uses data from the course readings and provides additional instruction in terminology and in planning analyses. Topics include MANOVA, discriminant function analysis, principal components analysis, exploratory factor analysis, and confirmatory factor analysis.
